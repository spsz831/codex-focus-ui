# codex-focus-ui v0.1.1

本版本聚焦可用性与阅读体验，主要新增：

- 自动同步 Codex 聊天为按天会话文件（`codex-auto-YYYY-MM-DD.jsonl`）
- 一键启动默认打开当天自动会话
- 会话删除功能（可在 UI 删除选中的会话文件）
- 提问/回答改为 Markdown 渲染，提升长文本可读性
- 新增快速滚动：按钮“到顶部/到底部” + 快捷键 `T` / `B`

## 使用建议

```bash
cd E:\WorkCodex\codex-focus-ui
npm run oneclick
```

如需同步指定日期：

```bash
powershell -ExecutionPolicy Bypass -File scripts/cfu.ps1 sync 2026-03-27
```
