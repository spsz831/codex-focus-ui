## v0.1.1 - 2026-03-27

- 自动同步 Codex 聊天为按天会话文件（`codex-auto-YYYY-MM-DD.jsonl`）。
- 一键启动默认打开当天自动会话。
- 会话删除功能（可在 UI 删除选中的会话文件）。
- 提问/回答改为 Markdown 渲染，提升长文本可读性。
- 新增快速滚动：按钮“到顶部/到底部” + 快捷键 `T` / `B`。
- 新增“导出勾选”能力，支持复选框选择后导出 Markdown。
- 新增“全选当前可见 / 取消全选当前可见”批量勾选能力。
- 新增本地控制面板 `cfu-control.cmd`（中英双语菜单、状态/启动/关闭/帮助）。
- 新增 `npm run ui` 与全局命令 `cfu-ui`，降低日常启动成本。

## v0.1.0 - 2026-03-27

- 建立 `codex-focus-ui` 基础项目与会话采集链路。
- Viewer 支持上一问定位、类型过滤、关键词搜索、书签、历史会话切换。
- 支持导出 Markdown 复盘清单。
- 引入配置文件、`doctor` 诊断命令与 `smoke-test` 自检脚本。
